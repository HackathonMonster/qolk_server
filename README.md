# Qolk - Server
Inform you about the humidity, temperature and alcohol of the wine which capped Qolk.

## POST
type: json
endpoint: /
```
{
  humidity: 54.03,
  temperature: 24.23
  alcohol: 50.00
}
```

## GET
type: json
endpoint: /
```
{
  humidity: 54.03,
  temperature: 24.23
  alcohol: 50.00
}
```
